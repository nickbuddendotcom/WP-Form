An API for creating and validating forms in WordPress

== Description ==

WP Form is an API for creating and validating forms in WordPress. It contains two classes:

* A class for programatically building a form's HTML.
* A class for validating a form's inputs

== Usage ==

This plugin is intended as a lightweight API for your own plugin or theme. For plugin documentation visit: https://github.com/nickbuddendotcom/WP-Form

== Installation ==

Installing WP Form can be done either by searching for "WordPress Plugin Template" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
